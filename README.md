# Patrones de Diseño - MDAS

Este proyecto tiene como objetivo la implementación de distintos patrones de diseño de software asignados semanalmente, utilizando el lenguaje de programación **Java**.

## Autores
* **Hugo Espejo Ramírez**
* **Rafael Luque Laplana**
* **José Luis García Corbacho**

## Estructura del Proyecto

El desarrollo se divide en bloques semanales, abordando un patrón de diseño específico en cada entrega:

*   **Semana 1:** Abstract Factory (En progreso)
*   **Semana 2:** Adapter
*   **Semana 3:** Bridge
*   **Semana 4:** Composite
*   **Semana 5:** Façade

## Semana 1: Abstract Factory

En esta primera entrega se ha implementado el patrón **Abstract Factory** para la gestión de menús (Semanal y Temporada) en diferentes contextos (Restaurante y Para Llevar), permitiendo la creación de familias de objetos relacionados sin especificar sus clases concretas.
